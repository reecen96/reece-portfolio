Daytona

September 19, 2021

I purchased an original Daytona arcade machine for restoration. Unfortunately a majority of what was once the arcade machine had been lost or used for parts. The remaining machine was the the frame, chair and some of the original electronics.

1. The first step to rebuild the machine was gutting the old electronics (which consisted of an old 240V power supply and a broken amplifier).




2. Once gutted the frame was de-rusted using sand paper & turpentine.



















































3. Metal polish was then use to clean up the chrome - used Autosol Metal Polish 75mL and it came out great




4. Next I cleaned up the chairs - reccomend using soap and water to avoid oxidising/bleaching the plastics



5. The seat was built with a subwofer under the frame, two speakers in the head rest and two floresent globes behind a glass pane. After testing, the globes were no longer working so I decided to go with some cheap 12v led strips.








































6. We then needed to design the cabinet. Keeping as close as we could to the original cabinet, we decided to go with 18mm MDF. We couldn't find any designs/dimentions for the original cabinet online so I had to find an original cabinet and measure myself. Very happy with the way the cabinet came out - consisted of two side panels, a back panel and a top panel. The cabinet walls attached perfectly to the original frame.



















































7. To finish the cabinet I used pictures of the original Dayton as a reference and photoshopped prints for the side panels. 18mm T-section rubber was used on the side length of the pannels. The back and top pannels were spray painted black.

Electronics

To run arcade roms/emulate Daytona I wanted to avoid using a PC, any operating system that required a mouse or keyboard or any UI that didn't look like an arcade. I decided to go with a Rasberry Pi running Retropie. Retropie is purpose built to emulate ROMS ie mame (arcade machine games like Daytona) and only required a few buttons to chose/interact with the UI. I used an old G27 wheel/pedals/gear stick and 5 buttons for controls.

























To fit the wheel into the dashboard the casing was removed and cut back. This was also done with the pedals & gear stick.
A casing was made for the gear stick using left over MDF and was wrapped with leather vynil. Metal checkerplate was cut and bent into the dashboard supported by an MDF frame. Checkerplate was also used to cover the gearstick housing. The buttons were inserted into the dashboard and button leds connected the 12v circut used for the chair led strips.

Tic Tac Foe

⚔ Tic-Tac-Foe ⚔
My project incorporates HTML, CSS, JS & Jquery to create an interactive game of tic-tac-toe. The project involves the mechanics & rules of tic tac toe, game scoring, timing, different board sized, animations, interactive menus, local storage and computer controlled players.


Main menu


3x3 box


5x5 box


Score board with local storage


How it was built

My project was built using HTML, CSS, JS & Jquery. Other external media applications were also used to create pixel graphics and animations.

How to play / Features

How to play / Features The application starts on a menu page giving the options to either play a game, game stats or multiplayer. Game stats retrieves data stored on local storage - the number of times each player has one and the total number of user clicks on the document. The multiplayer option links to a Firebase version on the project which can be used by multiple users simultaneously- similar to a multiplayer feature. If the Play game option is chosen you are able to play tic-tac-toe and the number of games each player has won is tracked at the bottom right of the screen. Each time an action is made there is an animation at the bottom of the screen. On the bottom left of the screen you are able to chose a Computer opponent (easy, medium or hard), chose the game square size (options are between 3x3 & 9x9) or play a timed game with the computer or another player.


Portable 12v speaker

The aim of this project is to build a speaker that is a louder and cheaper alternative to portable speakers such as the Megaboom and the JBL extreme. The speaker was built using hard rubish and second hand parts found online. The speaker has a 9 hour battery life, an output of 200 decebels and a total cost was less than $150.00. In comparison a Megaboom has a battery life of 14 hours, an output of 90 decebels and a total cost of $400.00. The speaker runs of two 12v 7.4mha batteries, running in parralel, powering a 820w car amplifier. The amplifier powers two efficient 6x9 Kenwood speakers and is turned on and off using an external switch. The battery voltage, used to determine the remaining power in the batteries, is monitored using an LCD voltage sensor and the batteries are recharged using a lipo battery charger. Accoustic foam was not needed in this build as the travel case used as the speaker housing already has foam padding.
