Plant Owl

description

The Arduino has been programmed in 'a c/c++' to post data to a server cyclically and water plants when their soil moisture falls below a certain %. On a specified interval the Arduino collects data (soil moisture, water level, temperature) and posts it as a JSON request to a Node.js/React/Mongodb server. This date is then access from a React front end where the data is graphed and analysed.

Fairy Floss

description

This application was created to connect tenants and Landlords together with an easy to use interface. Fairy Floss is an extension of the already existing Melbourne based Facebook page that allowed tenants to connect with potential roommates for lease transfers and room listings. Fair Floss allows for a user to List their a rental home or room, apply for a listing, create a user profile and manager their applications and listings. Implemented in the application is a Ruby Gem which shows listed properies on Google maps, user Authentication and an admin dashboard.

Tic Tac Foe

description

⚔ Tic-Tac-Foe ⚔
My project incorporates HTML, CSS, JS & Jquery to create an interactive game of tic-tac-toe. The project involves the mechanics & rules of tic tac toe, game scoring, timing, different board sized, animations, interactive menus, local storage and computer controlled players.

How it was built

My project was built using HTML, CSS, JS & Jquery. Other external media applications were also used to create pixel graphics and animations.

​ ​

How to play / Features

How to play / Features The application starts on a menu page giving the options to either play a game, game stats or multiplayer. Game stats retrieves data stored on local storage - the number of times each player has one and the total number of user clicks on the document. The multiplayer option links to a Firebase version on the project which can be used by multiple users simultaneously- similar to a multiplayer feature. If the Play game option is chosen you are able to play tic-tac-toe and the number of games each player has won is tracked at the bottom right of the screen. Each time an action is made there is an animation at the bottom of the screen. On the bottom left of the screen you are able to chose a Computer opponent (easy, medium or hard), chose the game square size (options are between 3x3 & 9x9) or play a timed game with the computer or another player.
